# Руководство Пользователя и Администратора

## Руководство Пользователя (Туриста)
Добро пожаловать в Интерактивный гид по ХМАО-Югре!

1. **Регистрация и Вход**: Откройте приложение, введите свои данные для создания профиля. При следующем входе приложение запомнит вас.
2. **Лента рекомендаций**: Сразу после входа вы увидите самые интересные и актуальные локации и события.
3. **Меню**: Нажмите на иконку списка в левом верхнем углу, чтобы открыть боковое меню. Вы сможете перейти в:
    - Объекты и Мероприятия
    - Списки Маршрутов
    - Городской Транспорт
    - Погода в реальном времени.
4. **Избранное**: В карточке любой локации вы можете нажать на сердечко, и она появится в вашем профиле во вкладке "Хочу посетить".

## Руководство Администратора
В качестве администратора вы имеете доступ к базе данных `yugra_guide.db` на сервере.
- Для добавления новых объектов, событий или маршрутов, вы можете использовать любой SQL клиент (например, DBeaver) и добавлять строки напрямую в таблицы `locations`, `events`, `routes`.
- Либо запустить скрипт `python seed.py`, который автоматически подтянет новые доступные места из OpenStreetMap.
